# java-central-tendency-calculator
I made a simple calculator to find the 3 main measures of central tendency, which are mean, median, and mode.

## Features

- User input handling
- Input validation
- Exception handling
- Bubble Sort implementation for median calculation

## Concepts Practiced

- Java methods
- Arrays (without relying on Array.sort() and other methods)
- Nested loops
- Exception handling
- Algorithm design
